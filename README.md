# Modular-Arduino-Based-Athletic-Vest
This fitness vest offers essential tracking—heart rate, steps, speed, and real-time goal feedback—combined with post-workout data storage via a local microSD card. Unlike traditional wearables, its open-enclosure design allows users to customize the device by adding Arduino-compatible sensors for advanced, personalized metrics. 

Materials:
•	Enclosed space (V5): battery/top/bottom pieces; two side holes for nuts/bolts securing
•	Nuts and bolts (for enclosure fastening)
•	M3 x7
•	M4 x2
•	Haimont 10L running Vest
•	Arduino + PCB “motherboard” assembly (PCB preserves Arduino pins through female headers)
•	SparkFun GPS module - Chip Antenna, SAM-M10Q (Qwiic)
•	SparkFun OpenLog module (Qwiic)
•	MicroSD card (recommended: SanDisk 32GB Ultra, FAT32, 32GB)
•	MicroSD USB card reader
•	MPU6050 Gyroscope/IMU
•	JST cable harness (2.54 mm JST + 2.0 mm JST LCD end)
•	Qwiic wires
•	ST7789 LCD screen + wrist case with Velcro
•	Power bank (INIU P41L-E2 class, 45W, 10,000 mAh)
•	Polar H10 chest strap + sensor (battery pre-installed)
